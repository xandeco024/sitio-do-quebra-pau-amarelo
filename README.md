## 🎮 Sobre o Jogo

**"Sítio do Quebra-Pau Amarelo"** mergulha os jogadores em um universo de **Battle Royale** inspirado na rica **literatura brasileira**. O jogo é uma homenagem carinhosa às obras clássicas do Brasil, combinando referências literárias com mecânicas de combate intensas em pixel art.

- Escolha entre **seis personagens icônicos**, cada um com habilidades únicas
- Embarque no lendário **"Trem das Onze"** de Adoniran Barbosa para iniciar sua jornada
- Envolva-se em batalhas intensas num **mapa inspirado no Sítio do Pica-Pau Amarelo**
- Celebre a literatura e a cultura brasileira enquanto luta para ser o **último herói de pé**

> Esta iniciativa da CGJ nos permitiu homenagear não apenas as obras literárias, mas também elementos da cultura brasileira, proporcionando uma experiência de jogo única.

## Como Jogar

### Download (Build pronta)

1. Acesse a página do jogo no **[itch.io](https://xandeco.itch.io/sitio-do-quebra-pau-amarelo)**
2. Clique em **"Download"** para obter o arquivo `1.0.1 Win.zip` (~36 MB)
3. Extraia o conteúdo do ZIP para uma pasta de sua escolha
4. Execute **`Sitio_do_Quebra_Pau_Amarelo.exe`** para iniciar o jogo

### Requisitos Mínimos

| Componente | Requisito |
|---|---|
| **Sistema Operacional** | Windows 7 ou superior |
| **Processador** | 1.2 GHz |
| **Memória RAM** | 2 GB |
| **Placa de Vídeo** | DirectX 10 + Shader Model 4.0 |
| **Armazenamento** | 200 MB livres |

## Abrindo o Projeto no Unity

### Pré-requisitos

- [Unity Hub](https://unity.com/pt/download) instalado
- Versão do **Unity Editor** compatível (verifique em `ProjectSettings/ProjectVersion.txt`)
- [Visual Studio](https://visualstudio.microsoft.com/) ou **VS Code** com extensão C#

### Passos

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/xandeco024/sitio-do-quebra-pau-amarelo.git
   ```

2. Abra o **Unity Hub**, clique em **"Add"** e selecione a pasta clonada.

3. Abra o projeto com a versão correta do **Unity Editor**.

4. Dentro da pasta `Assets/`, localize a **cena principal** e dê um duplo clique para carregá-la.

5. Pressione Play no editor para testar o jogo.
